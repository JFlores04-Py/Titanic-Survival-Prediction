# Titanic Survival Prediction

COGS 109 Final Project – Spring 2026

## Overview
Predicts Titanic passenger survival using Logistic Regression and KNN. Compares models with cross-validation, regularization, PCA, and bootstrap.

## Key Results
- Best KNN (k=21): **78.8%** test accuracy
- Logistic Regression: **79.3%** test accuracy
- Gender and passenger class are the strongest predictors
