# Titanic Survival Prediction

Predicting passenger survival on the Titanic using Logistic Regression and KNN.

- Best KNN (k=21): 78.8% test accuracy
- Logistic Regression: 79.3% test accuracy
- Gender and passenger class are the strongest predictors

## Files
- `titanic_project.ipynb` – Full analysis notebook
- `train.csv` – Dataset

## How to Run
1. Open `titanic_project.ipynb` in Jupyter Notebook
2. Run all cells

## Data Source
[Kaggle Titanic Competition](https://www.kaggle.com/c/titanic)
